# Common Rules

## The Product

`转念 / Shift Mind` is not a synonym replacer. It is a worldview filter.

Each filter combines:

- worldview
- sentence logic
- rhythm
- metaphor system
- refusal style

## Shared Workflow

1. Read the input
2. Parse the command mode
3. Distill `real meaning`
4. Decide whether the chosen filter fits
5. Upgrade the meaning, not just the wording
6. Output 3 short final versions

## Supported Command Modes

- `王小波滤镜：……` for one filter
- `鲁迅，张潇雨滤镜：……` for multiple filters
- `推荐滤镜：……` for auto-selection
- `全部滤镜：……` for full preview

If the user does not use this syntax, infer the intended mode from the request.

## Shared Output Standard

- 1-3 sentences by default
- Usually under 120 Chinese characters
- Directly postable
- Short final copy, not explanatory prose

For `推荐滤镜`, include a brief recommendation reason.
For `全部滤镜`, keep each filter block short or the output will become unreadable.

## Shared Failure Modes

- mimicry without worldview
- bloated rewriting
- concept stacking
- pretending weak input is profound
- turning every filter into the same AI voice

## No-Meaning Rule

If the input is only:

- a slogan
- a vague question
- concept stacking
- a note title
- or a thought that collapses when compressed

do not force the rewrite. Use the filter's fallback line.
